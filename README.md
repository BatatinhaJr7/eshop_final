E-Shop Brasil - Projeto

Conteúdo:
- app.py (Streamlit demo)
- utils/data_generator.py
- docker-compose.yml
- Dockerfile
- requirements.txt


Como testar localmente:
1. docker-compose up --build
2. Acesse http://localhost:8501

O protótipo demonstra:
- Inserção/edição/exclusão de clientes, produtos e pedidos.
- Geração de dados em massa para simular Big Data.
- Exibições e filtros básicos na interface.
